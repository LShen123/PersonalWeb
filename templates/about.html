About This Project

This project demonstrates the use of linear regression and logistic regression on the Iris dataset. The dataset contains measurements of iris flowers, and we use these measurements to predict different attributes of the flowers.

Linear regression is used to predict continuous values, such as petal width, while logistic regression is used for classification tasks, such as predicting the species of the iris flower.

The models are implemented using Python's scikit-learn library, and the results are displayed using a Flask web application.
